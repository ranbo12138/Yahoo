# Yahoo! 日漫翻译器

## v0.1.0 - 基础翻译功能

### 功能
- ⚙️ 设置页：配置 API Key、Base URL、Model
- 📝 翻译页：输入日文 -> 翻译成中文
- 📋 日志页：查看请求日志

### 使用方法
1. 下载 APK 并安装
2. 打开 APP，点击"设置"
3. 填入你的 OpenAI API Key 和 Base URL
4. 返回主页，输入日文，点击"翻译"

### 技术栈
- Kotlin
- Retrofit + OkHttp
- Coroutines
- Material Design 3
